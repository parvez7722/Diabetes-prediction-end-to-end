# Diabetes Prediction System (End-to-End ML Project)

## Overview
This project predicts whether a person has diabetes using machine learning.

## Tech Stack
- Python
- Scikit-learn
- FastAPI
- MLflow
- Docker

## Features
- ML pipeline with preprocessing
- Experiment tracking using MLflow
- REST API using FastAPI
- Dockerized application
- Logging and monitoring

## How to Run

### Using Docker
```bash
docker pull parvezmohammed381/diabetes-app:latest
docker run -p 8000:8000 parvezmohammed381/diabetes-app
